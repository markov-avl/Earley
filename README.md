## Нисходящий разбор с возратом

### Вход
Правила:
```
E -> T+E
E -> T
T -> F*T
T -> F
F -> a
```
Слово: `a+a`

### Вывод
```
      (q, 1, $, E#)
⊢ 1   (q, 1, $E₁, T+E#)
⊢ 1   (q, 1, $E₁T₁, F*T+E#)
⊢ 1   (q, 1, $E₁T₁F₁, a*T+E#)
⊢ 2   (q, 2, $E₁T₁F₁a, *T+E#)
⊢ 4   (b, 2, $E₁T₁F₁a, *T+E#)
⊢ 5   (b, 1, $E₁T₁F₁, a*T+E#)
⊢ 6.3 (b, 1, $E₁T₁, F*T+E#)
⊢ 6.1 (q, 1, $E₁T₂, F+E#)
⊢ 1   (q, 1, $E₁T₂F₁, a+E#)
⊢ 2   (q, 2, $E₁T₂F₁a, +E#)
⊢ 2   (q, 3, $E₁T₂F₁a+, E#)
⊢ 1   (q, 3, $E₁T₂F₁a+E₁, T+E#)
⊢ 1   (q, 3, $E₁T₂F₁a+E₁T₁, F*T+E#)
⊢ 1   (q, 3, $E₁T₂F₁a+E₁T₁F₁, a*T+E#)
⊢ 2   (q, 4, $E₁T₂F₁a+E₁T₁F₁a, *T+E#)
⊢ 4   (b, 4, $E₁T₂F₁a+E₁T₁F₁a, *T+E#)
⊢ 5   (b, 3, $E₁T₂F₁a+E₁T₁F₁, a*T+E#)
⊢ 6.3 (b, 3, $E₁T₂F₁a+E₁T₁, F*T+E#)
⊢ 6.1 (q, 3, $E₁T₂F₁a+E₁T₂, F+E#)
⊢ 1   (q, 3, $E₁T₂F₁a+E₁T₂F₁, a+E#)
⊢ 2   (q, 4, $E₁T₂F₁a+E₁T₂F₁a, +E#)
⊢ 4   (b, 4, $E₁T₂F₁a+E₁T₂F₁a, +E#)
⊢ 5   (b, 3, $E₁T₂F₁a+E₁T₂F₁, a+E#)
⊢ 6.3 (b, 3, $E₁T₂F₁a+E₁T₂, F+E#)
⊢ 6.3 (b, 3, $E₁T₂F₁a+E₁, T+E#)
⊢ 6.1 (q, 3, $E₁T₂F₁a+E₂, T#)
⊢ 1   (q, 3, $E₁T₂F₁a+E₂T₁, F*T#)
⊢ 1   (q, 3, $E₁T₂F₁a+E₂T₁F₁, a*T#)
⊢ 2   (q, 4, $E₁T₂F₁a+E₂T₁F₁a, *T#)
⊢ 4   (b, 4, $E₁T₂F₁a+E₂T₁F₁a, *T#)
⊢ 5   (b, 3, $E₁T₂F₁a+E₂T₁F₁, a*T#)
⊢ 6.3 (b, 3, $E₁T₂F₁a+E₂T₁, F*T#)
⊢ 6.1 (q, 3, $E₁T₂F₁a+E₂T₂, F#)
⊢ 1   (q, 3, $E₁T₂F₁a+E₂T₂F₁, a#)
⊢ 2   (q, 4, $E₁T₂F₁a+E₂T₂F₁a, #)
⊢ 3   (t, 4, $E₁T₂F₁a+E₂T₂F₁a, $)
```

## Восходящий разбор с возратом

### Вход
Правила:
```
E -> E+T
E -> T
T -> T*F
T -> F
F -> a
```
Слово: `a*a`

### Вывод
```
      (q, 1, #, $)
⊢ 2   (q, 2, #a, s$)
⊢ 1   (q, 2, #F, 5s$)
⊢ 1   (q, 2, #T, 45s$)
⊢ 1   (q, 2, #E, 245s$)
⊢ 2   (q, 3, #E*, s245s$)
⊢ 2   (q, 4, #E*a, ss245s$)
⊢ 1   (q, 4, #E*F, 5ss245s$)
⊢ 1   (q, 4, #E*T, 45ss245s$)
⊢ 1   (q, 4, #E*E, 245ss245s$)
⊢ 4   (b, 4, #E*E, 245ss245s$)
⊢ 5.2 (b, 4, #E*T, 45ss245s$)
⊢ 5.2 (b, 4, #E*F, 5ss245s$)
⊢ 5.2 (b, 4, #E*a, ss245s$)
⊢ 5.4 (b, 3, #E*, s245s$)
⊢ 5.4 (b, 2, #E, 245s$)
⊢ 5.3 (q, 3, #T*, s45s$)
⊢ 2   (q, 4, #T*a, ss45s$)
⊢ 1   (q, 4, #T*F, 5ss45s$)
⊢ 1   (q, 4, #T, 35ss45s$)
⊢ 1   (q, 4, #E, 235ss45s$)
⊢ 3   (t, 4, #E, 235ss45s$)
```
